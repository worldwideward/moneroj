  • {% trans "What exactly is Monero?" %}

    {% blocktrans %} Monero is a private, open-source cryptocurrency that runs on its own blockchain. Like Bitcoin, it can be mined using computer power to validate transactions and issue new coins. Unlike Bitcoin, Monero prioritizes privacy—your balances and transactions are hidden. Its decreasing inflation and scarcity also make it a strong store of value, similar to gold. {% endblocktrans %}

  • {% trans "How does Monero hide my money? How does it give me privacy?" %}

    {% blocktrans %} Monero uses advanced privacy technologies like RingCTs, Stealth Addresses, Bulletproofs, Dandelion++, and soon FCMPs. These tools hide the sender, receiver, amount, and even the IP address, ensuring full transaction privacy. {% endblocktrans %}

  • {% trans "Why do people say Monero is unstoppable?" %}

    {% blocktrans %} Monero supports atomic swaps with Bitcoin, allowing direct, private exchanges without intermediaries. Because these swaps rely on Bitcoin's blockchain, banning Monero would require censoring Bitcoin itself—an unlikely scenario—making Monero resistant to censorship. {% endblocktrans %}

  • {% trans "How does its inflation work? How are new coins created?" %}

    {% blocktrans %} Monero had no premine—all coins are earned by miners who secure the network. As supply increases, block rewards and inflation decrease. Monero, on the other hand, is private, fungible, faster, cheaper, and better suited for use as true digital money. {% endblocktrans %}
