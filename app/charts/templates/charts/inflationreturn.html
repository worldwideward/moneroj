About this chart?
Help about the chart
This chart shows the return on capital since inception for each privacy coin, plotted against its own reverse from inflation (x100). The x axis value means the number of years it would take to double the supply of the coin based on current inflation. This shows how Monero is a beast, having higher return at the same time as very low inflation. Both values are correlated and inversely proportional.
